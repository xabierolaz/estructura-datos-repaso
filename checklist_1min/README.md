# Checklist 1 minuto por tema

Esta es la unica capa activa de auto-ubicacion.

Objetivo:

- que el alumno se ubique rapido
- sin correccion automatica
- sin etiquetas impuestas por el sistema
- y decida por si mismo si un tema sale limpio, con dudas o no sale

## Regla de uso

1. Entra en [index.html](index.html).
2. Elige un tema teorico.
3. Intenta resolver las 3 anclas sin mirar apuntes.
4. Revela la solucion.
5. Marcate a ti mismo:
   - `Lo hago sin mirar`
   - `Lo saco con dudas`
   - `No lo saco`
6. Abre el bloque correspondiente de [resumen](../resumen/README.md).

## Regla de diseño

- Las anclas salen del contenido teorico principal resumido en `resumen/`.
- No se puntua ni se genera diagnostico automatico.
- El sistema no certifica nada.
- La honestidad la pone el alumno.
