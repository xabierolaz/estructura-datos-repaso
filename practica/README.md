# Practica para Spyder

Esta carpeta contiene ejercicios de practica alineados con el `resumen/`.

Objetivo:

- abrir un archivo `.py` en Spyder
- completar los `TODO`
- ejecutar `prueba.py`
- corregir errores hasta que las comprobaciones pasen

Tambien puedes usar esta capa desde Google Colab si has clonado el repo dentro del runtime, pero el formato base sigue siendo `.py`.

Estructura:

- cada bloque tiene su propia carpeta
- cada carpeta contiene:
  - `README.md`
  - `ejercicios.py`
  - `prueba.py`

Uso recomendado:

1. Estudia primero el bloque correspondiente en `../resumen/`.
2. Completa antes el cuaderno correspondiente en `../cuadernos/`.
3. Abre `ejercicios.py` en Spyder.
4. Completa las funciones o clases pendientes.
5. Ejecuta `prueba.py`.
6. Si falla un `assert`, vuelve al `cuaderno/` o al `resumen/` exacto y corrige.
