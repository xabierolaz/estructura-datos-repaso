# B01 - Practica en Spyder

Relacion con teoria: [B01 - Fundamentos de Python](../../resumen/01_fundamentos_python/README.md)

Objetivos:

- leer un fichero de texto
- transformar lineas en numeros
- usar `enumerate`
- construir texto con `f-strings`
- separar logica reutilizable de `main`

Archivos:

- `ejercicios.py`: completa las funciones
- `prueba.py`: ejecuta comprobaciones con `assert`
- `datos/numbers.txt`: fichero de entrada para practicar lectura
