# Inicio rapido para alumnado

Este repositorio se ha publicado a traves de GitHub Classroom.

## Si vienes desde la assignment

1. Acepta la invitacion de GitHub Classroom con tu cuenta personal de GitHub.
2. Abre el repositorio de tu entrega.
3. Lee primero [README.md](README.md).
4. Abre [cuadernos/README.md](cuadernos/README.md).
5. Empieza por [B01](cuadernos/01_fundamentos_python/cuaderno.ipynb).

## Si no aparece un IDE integrado

Eso no bloquea la practica.

Opciones recomendadas:

- abrir el cuaderno en Google Colab desde los enlaces de [cuadernos/README.md](cuadernos/README.md)
- o clonar el repositorio y trabajar en local con Jupyter y Spyder

## Ruta de estudio

La ruta correcta es:

1. `resumen/`
2. `cuadernos/`
3. `practica/`

No empieces directamente por `practica/` si no has cerrado antes el cuaderno del bloque.

## Requisitos minimos

- cuenta personal de GitHub
- acceso a Google Colab o entorno local con Python

## Cierre del bloque

Un bloque no se considera cerrado hasta que:

- completas el cuaderno
- haces `Restart & Run All`
- resuelves `ejercicios.py`
- ejecutas `prueba.py`
